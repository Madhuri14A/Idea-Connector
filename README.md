# Idea Connector

A semantic note-taking app with intelligent connection mapping using React, Node.js, and Neo4j.

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Visit `http://localhost:3000` in your browser.

## Database
Ensure Neo4j is running at `bolt://localhost:7687` with default credentials.
